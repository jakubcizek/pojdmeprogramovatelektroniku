## Vánoční stromeček s 25 RGB LED a webovým ovládáním
Tentokrát si v Pythonu oživíme modul vánočního stromečku 3D RGB Xmas Tree for Raspberry Pi s 25 adresovatelnými RGB LED. Světýlka nastavíme pomocí knihovny rgbxmastree a pomocí knihovny Tornado spustíme webový server s ovládáním světýlek skrze webový prohlížeč.
